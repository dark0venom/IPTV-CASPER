========================================
    IPTV CASPER - PORTABLE VERSION
========================================

This is the portable version of IPTV Casper.
No installation required!

HOW TO USE:
-----------
1. Extract this folder anywhere on your computer
2. Double-click "iptv_casper.exe" to run
3. Add your playlists and enjoy!

FEATURES:
---------
✓ No installation needed
✓ Run from USB drive or any folder
✓ All settings stored locally
✓ Native Windows controls (minimize, maximize, close)
✓ Support for M3U, Xtream Codes, and Local Files
✓ VOD and Series support
✓ Encrypted credential storage
✓ Floating window mode

SYSTEM REQUIREMENTS:
--------------------
- Windows 10/11 (64-bit)
- 4GB RAM minimum
- Internet connection for streaming
