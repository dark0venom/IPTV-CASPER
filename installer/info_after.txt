GETTING STARTED:
----------------

1. Launch IPTV Casper from:
   • Desktop icon (if created)
   • Start Menu > IPTV Casper
   • Or double-click any .m3u/.m3u8 file

2. Load Your IPTV Content:
   • Click "Add Playlist" to import an M3U file
   • Or click "Add Provider" for Xtream Codes
   • Enter your server details and credentials

3. Start Watching:
   • Browse categories in the sidebar
   • Click any channel to start playing
   • Use the PiP button for floating window mode

FLOATING WINDOW FEATURE:
------------------------
To use the Picture-in-Picture floating window:
1. Start playing any channel
2. Click the PiP button (floating window icon)
3. Click the pin button to enable always-on-top
4. Continue using other apps while watching!

KEYBOARD SHORTCUTS:
-------------------

F        - Fullscreen
M        - Mute/Unmute
←/→      - Seek backward/forward
↑/↓      - Volume up/down
Esc      - Exit fullscreen
